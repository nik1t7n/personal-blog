# blog

personal blog built with astro, react, and tailwind v4.

**live:** [blog.nik1t7n.work](https://blog.nik1t7n.work) *

## stack

- astro 5
- react 19
- tailwind css v4
- framer motion

## features

- markdown blog posts
- table of contents sidebar
- dark/light mode
- rss feed
- sitemap
- seo optimized

## dev

```bash
npm install
npm run dev
```

## build

```bash
npm run build
```

## add post

create a new `.md` file in `src/content/blog/`:

```md
---
title: "Your Post Title"
description: "Brief description"
pubDate: 2025-11-27
author: "Your Name"
tags: ["tag1", "tag2"]
---

Your content here...
```
